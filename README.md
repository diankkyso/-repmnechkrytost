# Weather Diary / Дневник погоды

GUI-приложение для ведения дневника погодных наблюдений. Позволяет добавлять, просматривать, фильтровать и удалять записи с сохранением в JSON.

**Автор:** Диана Шайхиева

## Функциональность

- **Добавление записей** с полями:
  - Дата (формат ДД.ММ.ГГГГ)
  - Температура (°C, поддерживаются дробные и отрицательные значения)
  - Описание погоды (текст)
  - Осадки (да/нет, checkbox)
- **Отображение** всех записей в таблице
- **Фильтрация:**
  - По конкретной дате
  - По минимальной температуре (показать записи выше указанного порога)
- **Валидация ввода:**
  - Проверка формата даты
  - Проверка, что температура — число
  - Проверка, что описание не пустое
- **Сохранение и загрузка** данных в JSON-файл
- **Удаление** выбранной записи

## Требования

- Python 3.8 или выше
- Tkinter (входит в стандартную библиотеку Python)
- pytest (для запуска тестов)
